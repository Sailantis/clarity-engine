Conditional Rendering Examples

Simple If Statement

{% if user.isLoggedIn %}

✓ Welcome back, {{ user.name }}!

{% endif %}

If-Else Statement

{% if stock > 0 %}

✓ Product is in stock ({{ stock }} available)

{% else %}

✗ Product is out of stock

{% endif %}

If-Elseif-Else Statement

{% if score >= 90 %}
Grade A: Excellent! (Score: {{ score }})
{% elseif score >= 75 %}
Grade B: Good! (Score: {{ score }})
{% elseif score >= 60 %}
Grade C: Satisfactory (Score: {{ score }})
{% else %}
Grade F: Needs improvement (Score: {{ score }})
{% endif %}

Comparison Operators

Age: {{ age }}

{% if age > 18 %}

✓ Over 18: {{ age > 18 }}

{% endif %} {% if age >= 18 %}

✓ 18 or older: {{ age >= 18 }}

{% endif %} {% if age == 21 %}

✓ Exactly 21!

{% else %}

Not 21 (age is {{ age }})

{% endif %}

Logical Operators

AND Operator

{% if user.isActive and user.role == 'admin' %} ADMIN ACCESS {% endif %}

OR Operator

{% if user.role == 'admin' or user.role == 'moderator' %} STAFF MEMBER {% endif %}

NOT Operator

{% if not user.isBanned %}

✓ Account is in good standing

{% else %}

⚠ Account is banned

{% endif %}

Complex Condition

{% if (age >= 18 and not user.isBanned) or user.hasParentalConsent %}

✓ User can access this content

{% else %}

✗ Access denied

{% endif %}

Ternary Operator

Status: {{ user.isActive ? 'Active' : 'Inactive' }}

Price: {{ discount > 0 ? price * (1 - discount) : price }} EUR {{ discount > 0 ? '(with ' ~ (discount * 100) ~ '% discount)' : '' }}

Null Coalescing

User nickname: {{ user.nickname ?? user.name }}

Custom message: {{ customMessage ?? 'No message provided' }}

Nested Conditionals

{% if user.isLoggedIn %}

Logged in as: {{ user.name }}

{% if user.hasNotifications %}

📧 You have {{ user.notificationCount }} new notifications

{% else %}

No new notifications

{% endif %}
{% else %}

Please log in to view your notifications

{% endif %}

Conditional Content Rendering

{% if products.length > 0 %}

Products Available ({{ products.length }})

    {% for product in products %}
  • {{ product }}
    • {% endfor %}
{% else %}

No products available

{% endif %}
