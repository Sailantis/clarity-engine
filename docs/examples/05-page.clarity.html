How It Works

The child template (this file) uses {% extends "05-layout" %} to inherit the structure from the parent layout. Each {% block blockName %} overrides the corresponding block in the parent.

Blocks that aren't overridden (like the footer and navigation) use the parent's default content, making it easy to create consistent page structures.
