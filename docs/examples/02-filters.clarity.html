Filter Examples

Filters transform data before output using the |> operator.

Text Filters

Original: {{ text }}
Upper: {{ text |> upper }}
Lower: {{ text |> lower }}
Capitalize: {{ text |> capitalize }}
Title: {{ text |> title }}
Trim: "{{ messyText |> trim }}"
Truncate: {{ longText |> truncate(50) }}

Number Filters

Original: {{ price }}
Number (2 decimals): {{ price |> number(2) }}
Number (0 decimals): {{ price |> number(0) }}
Rounded: {{ price |> round(1) }}
Absolute value: {{ negativeNumber |> abs }}

Date Filters

Default format: {{ timestamp |> date }}
Custom format (d.m.Y): {{ timestamp |> date('d.m.Y') }}
With time (H:i:s): {{ timestamp |> date('Y-m-d H:i:s') }}
Readable format: {{ timestamp |> date('F j, Y') }}

Array Filters

Join: {{ tags |> join(', ') }}
First element: {{ tags |> first }}
Last element: {{ tags |> last }}
Length: {{ tags |> length }} tags
Reverse: {{ tags |> reverse |> join(', ') }}
Sort: {{ numbers |> sort |> join(', ') }}

Filter Chaining

Multiple filters can be chained together:

Chain: {{ text |> lower |> trim |> capitalize }}
Complex chain: {{ description |> trim |> truncate(100) |> upper }}

Lambda Filters

Use lambdas with map, filter, and reduce:

Map (extract names): {{ users |> map(u => u.name) |> join(', ') }}
Filter (active only): {{ users |> filter(u => u.active) |> map(u => u.name) |> join(', ') }}
Map with filter (uppercase): {{ tags |> map(t => t |> upper) |> join(' | ') }}

Utility Filters

Default value: {{ emptyValue |> default('No value provided') }}
JSON encode: {{ data |> json |> raw }}
